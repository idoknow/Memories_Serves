Project Profile

项目资料

Memories Serves 是一个使用 Rust、Axum、SQLx 与 SQLite 构建的轻量 API 服务，主要用于保存图片 URL、记录上传时间戳、按 ID 顺序读取，并支持管理员删除和 WebDAV 备份。
